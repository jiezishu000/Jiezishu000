# find-skills — 智能技能发现

## 功能
自动搜索、发现、推荐 Claude Code 可用技能包。

## 技能源
- GitHub (关键词搜索 + Trending)
- Anthropic 官方技能库
- 社区推荐清单
- 本地自定义技能

## 使用方式
当用户说 "帮我找技能" / "发现新技能" / "有什么好的技能" 时自动触发。

## 发现流程
1. 理解用户需求场景
2. 搜索匹配技能
3. 评估质量（Stars、维护状态、兼容性）
4. 推荐 Top 3-5
5. 一键安装

## 交互规范
- 每个推荐说明：名称、用途、Stars、安装命令
- 优先推荐经过实战验证的技能
- 标注是否与现有技能冲突
