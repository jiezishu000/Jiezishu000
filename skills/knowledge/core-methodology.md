## 一、V3 统一工作台三段式

### 补环境差
- GitHub不通 → ghproxy.com 前缀
- npm不通 → npmmirror.com 镜像
- pip不通 → 清华源 pypi.tuna.tsinghua.edu.cn
- 所有代码必须"复制即运行"级别
- 禁止省略import/依赖声明/错误处理

### 断信息差
- 输出强制带：完整依赖安装命令 + 环境检测 + 测试命令 + 预期结果
- 标注"国内可用/需梯子"
- 国产AI替代方案：Kimi/DeepSeek替代ChatGPT/Claude

### 建循环
- 执行→失败→捕获错误→分类(8类)→修复→重试
- 三轮上限，超限切换策略
- 错误分类器：network_timeout / dependency_missing / git_clone_failed / npm_install_fail / pip_install_fail / memory_oom / rate_limit / dns_fail

## 五、导航站设计原则

- 布局：厂字型（ArcoDesign/飞书风格）— 左侧一级导航 + 顶部二级标签
- 导航逻辑：按用户任务流，不按内容类型
- 首页：定位句 + 目标选择器 + 搜索 + 热点卡片
- 内容标签：二维体系（难度×阶段）
- 起步：纯静态HTML，单文件，零依赖
- 冷启动：0-5000元，每月精确动作+预算
